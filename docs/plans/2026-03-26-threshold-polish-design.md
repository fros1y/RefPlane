# Threshold Control Polish Design

**Date:** 2026-03-26
**Status:** Approved

## Scope

Polish the restored multi-handle threshold control with:

- larger handles
- tap-to-select-then-drag interaction
- segmented track shading between thresholds

## Approved Behavior

- handles remain ordered and cannot cross
- tapping a handle selects it
- dragging moves only the selected handle
- the track is visually divided into ordered threshold regions

## Notes

- keep the existing threshold model unchanged
- preserve accessibility adjustable actions per handle
- keep the current value/color views using the same shared threshold control
